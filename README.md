<h1 align="center" style="font-weight: bold;">Flutter Meals</h1>
<p align="center">
<b>Flutter Meals</b> is a Flutter app to browse and filter meals by category, with favorites, dietary filters and detailed recipe screens.
</p>

## 📍 Features

- **Meal Categories**: Browse meals organized by category.
- **Filters**: Filter meals by diet (gluten-free, vegan, vegetarian, lactose-free).
- **Favorites**: Save your favorite meals.
- **Recipe Detail**: Step-by-step instructions and ingredient list.
- **Smooth Navigation**: Tab and stack navigation throughout the app.

## 💻 Technologies

- Flutter / Dart
- Raleway & RobotoCondensed (custom fonts)

## 💡 How to Use

1. Clone the repository:
```bash
   git clone https://github.com/dm-moro/FlutterMeals.git
```

2. Navigate to the project directory:
```bash
   cd FlutterMeals
```

3. Install dependencies:
```bash
   flutter pub get
```

4. Run the app:
```bash
   flutter run
```
